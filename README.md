## Local Setup

1.  fork and clone
2.  `yarn`
3.  `touch .env.development`
4.  Add these entries in the `.env.development`

    ```
    AUTH0_DOMAIN=xxxxxxxxxxxx.auth0.com
    AUTH0_CLIENT_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxx
    ```
5.  `yarn develop`
